## 扩展插件用处

更改网页背景色

## 使用

1. url: `chrome://extensions`
2. 打开开发者模式
3. 加载以解压扩展程序或者直接下载`myExtension.crx`文件，然后拖入到 `chrome` 浏览器 `chrome://extensions/` 页面内
